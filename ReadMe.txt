There are three python files in this project, including create_xlsx.py, main.py, and scraping_function.py.
Each of them have specials functions. Also you need the latest Chrome webdriver which available in the Driver Folder.

In order these script works, first you have to run main.py file which contains the core of the program.
The scraping_function.py will retrieve all the necessary fields and crete_xlsx.py will receive the data to
create xlxs file for each categories, such as ASN, POC, Net, Net resource, and Org.
